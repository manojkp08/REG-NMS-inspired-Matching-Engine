# GoQuant Matching Engine

A high-performance cryptocurrency matching engine implementing price-time priority and REG NMS principles.

## Features

- ✅ Price-time priority matching
- ✅ Market, Limit, IOC, FOK order types  
- ✅ REST API + WebSocket feeds
- ✅ Real-time trade execution
- ✅ Order book management
- ✅ Comprehensive testing

## Quick Start

### Using Docker (Recommended)

```bash
# Start the engine
docker-compose up -d

# Run demo
python scripts/demo_script.py
